変数を使った演算とprint関数による出力

print関数は、文字列や数値を出力します。

print("文字列")  　#　文字列（文字の集まり）を出力するとき
print(数値)           #　数値や数式の演算結果を出力するとき

test.pyではprint関数の（）の中に文字列や数値を入力しましたが、【例題1】では変数を使って演算します。
変数とは数値や文字などを格納しておく箱のようなものです。変数に値を代入し,演算した結果を出力します。

【例題1】変数を使った演算（ファイル名はrei1.py）

変数に名前をつけるときは以下のルールがあります。name や num など分かりやすい名前をつけるとよいです。

1) 1文字目は英文字かアンダーバー
2) 2文字目以降は英数文字、アンダーバー
3) 予約語は使用できない
4) 大文字と小文字は区別される

予約語とはあらかじめ関数名や演算子として登録されているもので、例えば次のようなものがあります。
関数や演算子はエディタ上で色分けされるので予約語かどうかの区別は簡単です。

【主な予約語 = 変数名として使えないもの】
and del for is raise assert elif from lambda return break else global not try class except if or while continue exec import pass yield def finally in print など

演算子のまとめ（算術・代入・文字列演算）

（1）算術演算子 

a + b         # 加算
a - b         # 減算
a * b         # 乗算
a / b         # 除算
a % b         # a を b で割った余り
a ** b        # a の b 乗
a // b        # 切り捨て除算

（2）代入演算子（++や--はサポートされていないので、a+=1　やa-=1　を用います）

a = b         # a に b を代入する
a += b        # a = a + b に同じ
a -= b        # a = a - b に同じ
a *= b        # a = a * b に同じ
a /= b        # a = a / b に同じ

（3）文字列演算

a + b         # 文字列 a と 文字列 b を連結します
a * n         # 文字列 a を n 回繰り返します
